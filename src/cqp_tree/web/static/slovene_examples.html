Examples

If you have reached this page, you are probably wondering what kind of patterns you can search for with CQP/Tree. In this page, we have gathered a few examples in depsearch and Grew-match, Unless otherwise specified, you can copy-paste each example into the web interface and run them on a variety of UD-annotated Slovene corpora, such as SUK 1.1. Gigafida 2.2. With some adjustments, you can search for similar structures on any dependency-annotated corpus available on CLARIN.SI noSketch Engine or even on other CQL-compatible corpus search systems.
